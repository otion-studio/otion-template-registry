# Meet the AI agent

The bot button in the bottom-right corner opens an agent that can read and edit this workspace — write pages, fill databases, research on the web, and tidy up after you.

## What to try first

- Ask it to *"summarize my Inbox into three priorities"*.
- Ask it to *"add a Groceries database with the usual columns"*.
- Save a prompt you run often as a routine, like this one:

<!-- otion:routine {"prompt":"Review the First Steps database. For every row still in Todo, suggest the single quickest action to complete it.","label":"Nudge me along","collapsed":"true"} -->

## How it works

Bring your own API key in **Settings → AI**. The agent works on your local files — it sees exactly what you see and nothing leaves your machine except the model calls you configure.
