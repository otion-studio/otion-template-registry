# Everything is a file

Open this workspace's folder in your file manager. What you'll find:

- Every page is a `.md` file you can read in any text editor.
- Every database is a `.db.json` file with your rows in plain JSON.
- Sub-pages live in a folder named after their parent.
- Automatic snapshots live in `backup/` — restore any of them from Settings.

## Why it matters

- **No lock-in.** Your notes outlive any app, this one included.
- **Sync your way.** Git, Dropbox, iCloud, a USB stick — it's just a folder. Or use Otion's own collaborative sync server when you want live multi-user editing.
- **Agent-friendly.** Any AI tool that can read files can work with your workspace.
